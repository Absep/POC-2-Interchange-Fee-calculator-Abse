# [Pay-01] Interchange Fee Architecture Terminus

A high-performance interactive financial infrastructure terminal built for the **Real Rails Intelligence Library**. This system models transactional processing frictions, global routing corridors, and network take-rates.

## 🛠️ Tech Stack
- **Frontend:** Next.js 14+ (App Router), TypeScript, Tailwind CSS
- **Backend:** Python FastAPI, Pandas, DuckDB relational analytics pipeline

## 🛰️ Monitored Sovereign Sources
- Consumer Financial Protection Bureau (CFPB)
- Bank for International Settlements (BIS) CPMI / Red Book
- European Central Bank (ECB) Data Portal

## 🚀 Execution
1. **Backend Engine:** Run `python main.py` inside `/backend` (with active virtual environment)
2. **Frontend Dashboard:** Run `npm run dev` inside `/frontend`
